# Excel Sales Dashboard Project

## Overview

This project is an Excel-based sales analysis dashboard. It uses raw sales data to generate insights through formulas, pivot tables, and visual charts.

---

## File Structure

1. Raw_Data
   Contains the original dataset including transactions, customers, products, and sales details.

2. Analysis
   Includes key performance indicators (KPIs) calculated using Excel formulas such as total revenue, total orders, average order value, and unique customers.

3. Pivot
   Contains pivot tables used for summarizing data by region, category, customer, and time.

4. Dashboard
   Displays a visual summary of the analysis using KPI cards, charts, and slicers for interactive filtering.

---

## Features

* Data cleaning and table formatting
* KPI calculations using formulas
* Pivot table analysis
* Interactive dashboard with slicers
* Charts for visual representation of trends


## Tools and Functions Used

* SUM, AVERAGE, COUNTA
* SUMIFS, IF
* FILTER
* TODAY, NOW, DATEDIF, EOMONTH
* Pivot Tables
* Charts (Bar, Pie, Line)
